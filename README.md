# MultilevelMonteCarloMethods

This is a code repository containing all of the MATLAB M-functions used to produce results for my Final Year Project, titled 'Multilevel Monte Carlo methods for Options Pricing'. 

The M-functions mlmc.m, mlmc_plot.m, mlmc_test.m, mlmc_test_100.m are all downloaded from Giles' webpage https://people.maths.ox.ac.uk/gilesm/mlmc/. 

The M-functions america_multilevel.m and america_test.m have been written by me to price American options in a MLMC context.
